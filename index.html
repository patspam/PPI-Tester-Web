PPI::Tester - the Ajax version

On-the-fly lexing/dumping of Perl source code using Adam Kennedy's PPI module

During the Padre hackathon at YAPC::Europe 2008 Adam showed me the wxPerl app he wrote to stay one step ahead of Damian Conway during mind-bending YAPC keynotes (PPI::Tester). For fun, I decided to make a web-based version.

You can use this as a web service by sending a request to ppi.cgi with your perl code in the src query param, e.g. ppi.cgi?src=print%20%22Hello,%20World!%22
